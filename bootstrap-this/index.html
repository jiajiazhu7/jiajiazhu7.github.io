Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and (optinally) JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Easy

Bootstrap is easy to use. Being easy to get started with is probably the first quality which makes Bootstrap very appealing.

Easily Customizable

It is very easy to make good looking responsive websites using bootstrap. The default look of bootstrap components can be easily customized
